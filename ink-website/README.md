# Ink Website

Marketing landing page for [Ink](https://github.com/Cloverings1/Ink) — floating notes for macOS.

## Stack

- **Vite** + **React 19** + **TypeScript**
- **Tailwind CSS v4** (`@tailwindcss/vite`)

## Run locally

```bash
cd ink-website
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```
